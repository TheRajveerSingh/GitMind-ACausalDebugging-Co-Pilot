Before pushing, always check where you are: git branch
1. Push updated code to your personal repo
git checkout main
git add .
git commit -m "Your message"
git push

Because:
main -> origin/main
...............
2. Push updated code to the organization repo branch
git checkout rajveer-gitmind
git add .
git commit -m "Your message"
git push

Because:
rajveer-gitmind -> capgemini-new/rajveer-gitmind
